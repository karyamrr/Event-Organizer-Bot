# Проєкт: Бот-організатор подій

### Мета проєкту
Розробка консольної програми на Python, яка допомагає студенту вести розклад занять, іспитів та зустрічей. Програма дозволяє не лише зберігати події, а й уникати накладок у часі.

### Логіка та алгоритм роботи
Програма працює за таким принципом:
1. Під час старту дані зчитуються з JSON-файлу.
2. Користувач вводить команди через консоль (додати, показати, видалити тощо).
3. При додаванні нової події бот автоматично перевіряє, чи не зайнятий цей час іншою справою.
4. Після кожної дії список оновлюється та зберігається у файл.

### Структура даних
Всі події зберігаються як список словників. Кожна подія має такі ключі:
- назва (name)
- дата (date)
- час (time)
- категорія (category)

### Інструкція зі запуску
1. Встановіть Python версії 3.10 або вище.
2. Завантажте файл `bot.py` та запустіть його через термінал командою: `python bot.py`.

### Приклад роботи програми
- Користувач: додати
- Бот: Введіть дані події (назва: Лекція, дата: 2024-05-20, час: 10:00).
- Бот: ✅ Подію успішно додано!
- Користувач: сьогодні
- Бот: Виводить список справ на поточну дату.
